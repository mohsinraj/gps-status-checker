Paste one URL per line. The tool will attempt to check whether a site is alive, indexable (best-effort), and likely to provide dofollow links (best-effort).

# URL Alive HTTP Indexed (meta check) Likely Dofollow Notes
How this tool works (important)
  1. Alive: Performs a fetch to the URL (optionally via a CORS proxy). If fetch succeeds the site is considered reachable.
  2. Indexed: This does a best-effort check in the page markup for <meta name="robots"> or noindex. This is not a definitive Google index check because Google search results are not directly queryable from the browser due to CORS and rate-limits.
  3. Dofollow: Scans anchor tags on the page for rel="nofollow" and scans meta robots for nofollow. If no signals are found, the page is reported as likely dofollow. This is also a best-effort heuristic.

To get full, authoritative index status, run a server-side check calling Google, use an SEO API (Ahrefs/Moz/SEMrush), or check manually with site:example.com "page-url" in Google Search Console or Google.
